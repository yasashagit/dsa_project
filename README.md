## **ğŸ“˜ Project Overview**

This project explores the design and implementation of fundamental **data structures** and **algorithms** in Python.  
It includes hands-on implementations of:

- Arrays  
- Lists  
- Linked Lists  
- Stacks  
- Queues  
- Binary Trees  
- Binary Search Trees  
- Balanced Trees (AVL)  
- Hash Tables  

Additionally, the project includes small essential algorithms (sorting, searching), visual demonstrations,  
Graphviz-generated tree structures, step-by-step explanations, and time/space complexity analysis.

Each data structure is implemented inside the `dsa/` module as a Python class.  
The Jupyter Notebook (`notebook/`) demonstrates how they work through examples, diagrams, and short scenarios.
